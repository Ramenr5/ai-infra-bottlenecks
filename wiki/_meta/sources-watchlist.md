# Sources watchlist

The curated list of **credible, highly-influential AI/tech sources** the daily **AI Narrative & Findings Scout** monitors for new high-signal content (product/research announcements, important papers, perspective essays). Edit freely — the scout reads this file each run, so adding/removing a source needs no code change.

Quality bar: sources that **move the narrative or surface non-obvious signal**. Not general tech press. Each entry: what it's good for + how to find its latest.

---

## Scout rules of thumb

- **Signal over volume:** 3-6 genuinely notable items max per run; skip rehashes/noise. Quiet days = no message.
- **Always attach perspective:** every item gets a one-line "what it means for the thesis" (value-capture / compute-supply / power / regime). A finding with no thesis angle probably isn't worth surfacing here.
- **Flag strong claims for stress-test:** if a piece makes a falsifiable thesis-relevant claim, tag it for [[Thesis stress-tests]].
- **Dedup:** skip anything already in [[ingested-sources]].
